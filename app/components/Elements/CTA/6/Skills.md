# Required Skills

To implement this component, you should have knowledge of:

## Core Technologies

- **Vue 3**: Composition API, script setup, template syntax, reactivity system
- **Nuxt 3**: File-based routing, auto-imports, composables, server-side rendering
- **Nuxt UI v4**: Component library usage (UButton, NuxtImg, USlideover, UCard, etc.)
- **Tailwind CSS v4**: Utility-first CSS framework, responsive design, dark mode, custom utilities

## Additional Skills

- **TypeScript**: Type definitions, interfaces, type safety (optional but recommended)
- **Responsive Design**: Mobile-first approach, breakpoints, flexible layouts, viewport units
- **Modern CSS**: Flexbox, Grid, gradients, positioning, transitions, animations
- **Accessibility**: ARIA attributes, semantic HTML, keyboard navigation, screen readers
- **Performance**: Image optimization, lazy loading, code splitting
- **Component Architecture**: Single File Components (SFC), props, emits, slots

## Best Practices

- Clean, readable, and maintainable code
- Consistent naming conventions
- Proper component composition
- Performance optimization
- Accessibility compliance
- Cross-browser compatibility
