# landinuxt CLI

Install LandiNuxt landing sections into any Nuxt project.

```bash
npx landinuxt add hero/simple
npx landinuxt add features/bento
npx landinuxt list
```

## Options

| Flag | Description |
|------|-------------|
| `--out <dir>` | Output directory (default: `app/components/`) |
| `--name <file>` | Output filename |
| `--dry-run` | Print path without writing |
| `--site <url>` | LandiNuxt base URL (default: https://www.landinuxt.com) |

## Publish

```bash
cd packages/cli
npm publish
```
